# img2pureDiv 
利用 `canvas` 的 `getImageData` 获取图片每个像素点的 rgba 值。

利用多重 `box-shadow` ，每个像素点的值由一个一重 `box-shadow` 实现。将图片转换为单 div 。

[演示Demo](http://chokcoco.github.io/demo/img2div/html/)
